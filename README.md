# WorseGI

适用于Linux + Wayland平台(X11应该也可以?)的原神自动化工具

> [!NOTE]
实现思路来自[BGI](https://github.com/babalae/better-genshin-impact)与其附属/参考项目

> [!CAUTION]
此工具并不会修改游戏文件/运行时内存；使用本工具产生的一切后果由使用者自行承担，开发者不做任何担保.

## 前置条件/依赖

- `uinput`权限: 用于创建虚拟设备并模拟输入事件
- `gstreamer(gst-plugin-pipewire)`: 用于获取图像数据
- `opencv`: 用于图像相关任务[WIP]
- `onnxruntime`: 用于推理[WIP]
